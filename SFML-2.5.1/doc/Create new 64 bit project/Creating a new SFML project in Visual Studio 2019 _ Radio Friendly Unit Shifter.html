Creating a new SFML project in Visual Studio 2019

December 15, 2019 / Syaiful Nizam Yahya

A simple tutorial for creating SFML project in Visual Studio 2019.

Objective: To create a new SFML project.

Prerequisite

  1. SFML.
  2. Visual Studio 2019.

Time to complete: 30 minutes.

Instructions

  1. In Visual Studio 2019, Create a new project -> Empty Project. Give the project a name.
  2. Copy extracted SFML into the project solutions folder.
  3. Add a new main.cpp file by right click the project file and choose Add -> New item.. Enter main.cpp as the new file.
  4. Paste this texts as the contents of main.cpp.
    #include <SFML/Graphics.hpp>

int main()
{
    sf::RenderWindow window(sf::VideoMode(200, 200), "SFML works!");
    sf::CircleShape shape(100.f);
    shape.setFillColor(sf::Color::Green);

    while (window.isOpen())
    {
        sf::Event event;
        while (window.pollEvent(event))
        {
            if (event.type == sf::Event::Closed)
                window.close();
        }

        window.clear();
        window.draw(shape);
        window.display();
    }

    return 0;
}
  5. Configure the project properties as shown. As of writing, the latest SFML is 2.5.1. Modify the SFML folder if required.
  6. Make sure the target is x64.
  7. Compile and run. You should get this window.
